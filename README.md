# Quant Foundations

## Objective

Develop a structured foundation in quantitative finance by building data-driven workflows for market analysis and basic strategy development.

---

## Scope (30 Days)

### Week 1 — Data & Programming Foundations

* Python fundamentals for data handling
* Numerical computing with NumPy
* Data manipulation using Pandas
* Data visualization with Matplotlib
* Working with structured market data (CSV / bhavcopy)

**Output:**

* Clean data ingestion pipeline
* Basic exploratory data analysis (EDA)

---

### Week 2 — Market Data Analysis

* OHLC data interpretation
* Return calculations (simple returns)
* Volatility estimation (standard deviation)
* Correlation analysis across assets
* Moving averages and basic indicators (RSI)

**Output:**

* Feature-engineered dataset
* Analytical visualizations (trend, volatility, correlation)

---

### Week 3 — Strategy Development & Backtesting

* Rule-based trading strategy design
* Entry/exit signal generation
* Backtesting on historical data
* Performance metrics:

  * Profit & Loss (PnL)
  * Win rate
  * Maximum drawdown

**Output:**

* End-to-end backtested trading strategy
* Performance evaluation framework

---

### Week 4 — Packaging & Positioning

* Repository structuring and documentation
* Project presentation (GitHub + README)
* Resume integration of projects
* Basic outreach (applications / networking)

**Output:**

* Two complete projects:

  1. Market Data Analysis (EDA)
  2. Strategy Backtesting System
* GitHub-ready portfolio
